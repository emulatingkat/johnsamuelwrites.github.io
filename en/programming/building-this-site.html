Building this Site

John Samuel

After quite a long period without blogging, I am building this site to discuss about my work, my research and some personal experiences. Like programming, this entire site is built using the version control since it gives me the ability to track the progress and also have a glance on the past.

Currently the whole site is built only using HTML and CSS. I am not using any blogging platform like Jekyll. The site is aimed to have a responsive and minimalistic design, multilingual, human and machine-readable (with the help of semantic web technologies).
