Inkscape Options

  • Document Properties (Shift+Ctrl+D): Portrait or Landscape
  • Rectangles and Squares (F4)
  • Circles, Ellipses and Arcs (F5)
    • Start and End
  • Freehand Lines (F6)
    • Smoothening
  • Bezier Curves and Straight Lines (Shift+F6)
  • Filters
  • Extensions
    • Modify Path->Perspective
    • Render->L-System (for making fractals)
  • Zoom in (+) /Zoom out (-)
  • Export (as svg)
  • Edit Nodes (F2)
    • Break Path at selected nodes
    • Join Nodes (Shift+J)
    • Delete segment between two non-endpoint nodes
  • Break Apart Object (Path->Break Apart (Shift+Ctrl+k)
  • Union (Path->Union)
  • Flip Horizontal (H)
  • Flip Vertical (V)
  • Object to Path (Shift+Ctrl+C) (Path->Object to Path)
  • Ruler
  • Scale Multiple Objects (Shift+Click)
  • Guides
    • Edit->Create Guides around the page
    • Show/Hide Guides (Alt+|)
    • Object to Guides (Shift+G)
  • Spirals (F9)
    • Turns
    • Divergence
    • Inner Radius
  • Align and Distribute (Shift+Ctrl+A)
    • Align
      • Relative to [Page]
      • Center on horizontal axis
      • Center on vertical axis
  • Grid
    • Show/Hide Grid (#)
  • Scale Object
    • Select Object(s)
    • Object->Transform
    • Scale (Width and Height) and Apply
