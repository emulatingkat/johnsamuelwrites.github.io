Exercise 1.2

Most of the time, we work with CSV (comma-separated values) files. A CSV file consits of one or more lines and each line has one or more values separated by commas. One can consider every line as a row and every value in a row as a column value. The first row is sometimes used to describe the column names.

Copy the file pl.csv to your current working directory and use the following code to parse the csv file.

import numpy as np
dataset = np.loadtxt("pl.csv", dtype={'names': ('name', 'year'), 'formats': ('U100', 'i4')},
   skiprows=1, delimiter=",", encoding="UTF-8")
print(dataset)

CSV support in numpy is different from Python's default CSV reader because of its capability to support the data types.

Copy the file pl.tsv to your current working directory and use the following code to parse the tsv file.

import numpy as np
dataset = np.loadtxt("pl.tsv", dtype={'names': ('name', 'year'), 'formats': ('U100', 'i4')},
   skiprows=1, delimiter="\t", encoding="UTF-8")
print(dataset)

Note the changes in the above file. A TSV file is a tab-separated file, i.e., the column values are separated by a tab (\t).

Exercise 1.3

Most of the external data sources may provide their data in JSON files. Our next exercise is to parse JSON files. Copy the file pl.json to your current working directory and use the following code to parse the JSON file.

from pandas.io.json import json_normalize
import pandas as pd
import json

data = json.load(open('pl.json'))
dataframe = json_normalize(data)
print(dataframe)

Exercise 1.4

In this exercise, we will take a look at how to download data from external data sources using special query interfaces. Take for example, above data was obtained from Wikidata query interface with the query examples. See the screenshot given below.

Given below is the code to read data from an external data source. Use this url and replace ".." by the previous URL value.

import urllib
import json

url = "..."
response = urllib.request.urlopen(url)
responsedata = json.loads(response.read().decode('utf-8'))

array = []

for data in responsedata['results']['bindings']:
  array.append([data['year']['value'], data['languageLabel']['value']])
dataframe = pd.DataFrame(array, columns=['year', 'languageLabel'])
